# NH3-decomposition-kinetics
Kinetic modeling and Langmuir–Hinshelwood fitting for NH3 decomposition over promoted Co–Ni catalysts.

This repository contains the Python code used for fitting the semi-empirical Langmuir–Hinshelwood kinetic model for ammonia decomposition.

## File

H2_poisoning_LH_model_fit.py  
Python script used to fit the kinetic parameters and analyze hydrogen poisoning.

## Requirements

Python packages required:

numpy  
scipy  
pandas  
matplotlib  

## Usage

Run the script:

python H2_poisoning_LH_model_fit.py

The script performs nonlinear fitting of the kinetic model parameters and generates parity plots
